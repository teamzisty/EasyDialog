# EasyDialog

## 概要

Windowsダイアログを簡単に表示してくれるアプリです。

オレオレ証明書を使用しているので、100%ウイルス認定されるでしょう★

## 内容(中身)

中には
```
■
┣EasyDialog.exe
┣EasyDialog-forLegacyOS.exe
```
が入っています。

2つの違いは対象フレームワークで、

前者は ".Net Framework4.7.2"

後者は ".Net Framework2.0"を使用しています。

なお、後者のサポートは切れているので気をつけてください。

## 動作確認済み環境

EasyDialog.exe
- Windows7-32&64bit
- Windows10-64bit
- Windows11-64bit

EasyDialog-forLegacyOS.exe
- Windows7-32&64bit
- Windows10-64bit
- Windows11-64bit
